# Magisk Module Repo Builder

Clone the repo

```bash
git clone https://github.com/Googlers-Magisk-Repo/Magisk-Module-Repo-Builder.git
```

```bash
cd Magisk-Module-Repo-Builder
# Then, follow the prompts
bash build.sh
```

## Building on mobile

You need [Termux][termux] to build on mobile
